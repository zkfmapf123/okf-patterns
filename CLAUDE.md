# CLAUDE.md — Global Knowledge Base 규칙

## 목적
모든 세션에서 지식을 OKF 형식으로 일관되게 다루기 위한 개인 전역 규칙.
상세 형식·절차·예시는 "okf-knowledge-base" 스킬에 있으며,
이 문서는 항상 지켜야 할 불변 원칙과 스킬 발동 안내만 담는다.

## 스킬 발동 안내 (언제 OKF 스킬을 쓰는가)
다음 상황에서는 "okf-knowledge-base" 스킬을 로드해 그 규칙과 절차를 따른다.
- 대화에서 나온 지식(결론·절차·결정·참고자료)을 KB에 **저장/갱신**할 때
- KB에 있을 법한 내용을 **검색·참조**할 때
- 기존 KB 문서를 **수정**해야 할 때
판단이 애매하면 스킬을 로드해 규칙을 확인한 뒤 진행한다.

## 저장 위치 (불변)
- KB 루트: `~/.claude/kb/`
- `~/.claude/kb/common/` — 팀 공용 KB. 직접 수정·삭제 금지. **읽기(참조)만 허용.**
- `~/.claude/kb/local/`  — 개인 로컬 KB. 모든 신규 저장의 기본 위치.
- 별도 지정이 없으면 저장은 **항상 `~/.claude/kb/local/` 에 한다.**

## 저장 판단 (불변)
- 대화 내용은 **기본적으로 저장하지 않는다.**
- 지식성 내용이 있었던 경우에 한해, 대화 마지막에 사용자에게 저장 여부를 물어본다.
- 사용자의 명시적 요청이 없으면 저장하지 않는다.

## 안전 규칙 (불변)
- 자격증명·시크릿·API 키 등 민감정보는 본문에 저장하지 않는다.
- `~/.claude/kb/common/` 은 어떤 경우에도 직접 수정·삭제하지 않는다. (승격 PR로만 갱신)
